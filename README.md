## Data Structure & Algorithm
https://github.com/blair-cheng/CS214
Set as a private repository per Northwestern professor's policy. Please reach out for access (current students excluded).

**Language:** DSSL2 (typed teaching language built on Racket)  
**Assignments:** 4 + final project · **Lines:** ~2,200

Bottom-up construction of every fundamental data structure from scratch: linked lists, stacks, queues, association lists, hash tables (separate chaining), adjacency matrix graphs, DFS, and a capstone trip planner integrating all of them with Dijkstra's shortest path algorithm and binary heap priority queues.

**Highlights:**
- Trip Planner final project: full geographic route planning system with six internal data structures (graph, hash tables, heaps), Dijkstra's algorithm, and three query types (locate, route, nearby)
- Comprehensive testing: 10–20+ edge-case tests per assignment, including boundary conditions, error handling, and worst-case scenarios
